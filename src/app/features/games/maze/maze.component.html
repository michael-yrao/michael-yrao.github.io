Same maze, same start and goal — BFS explores level by level and provably finds the shortest path; DFS commits to one corridor at a time and returns whatever path it stumbles into. Run both on a few mazes and watch the gap.

How it works

Phase 1 — carving (randomized DFS)

Start at the top-left cell. Walk to a random unvisited neighbor and knock down the wall between you. Dead end? Backtrack to the last cell with an unvisited neighbor. When the stack empties, every cell is reachable and there is exactly one path between any two cells — a perfect maze. This is the same DFS + backtracking shape as exploring an island in Max Area of Island.
